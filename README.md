# MEM_VIZ.SYS — Interactive Memory Management Visualizer

**A high-fidelity, interactive simulator for Operating Systems (OS) concepts.**

## 🚀 Live Demo

**[Click here to run the Visualizer](https://www.google.com/search?q=https://arunavaghosh42428.github.io/MEM_VIZ.SYS-Interactive-Memory-Management-Visualizer/)**

## 🛠 Features

* **Dynamic Paging:** Real-time frame allocation for multiple processes (PROC_A, PROC_B, PROC_C) with virtual-to-physical address translation.
* **Segmentation:** Interactive memory mapping for CODE, DATA, HEAP, and STACK segments with adjustable sizes.
* **Virtual Memory Hierarchy:** Visual representation of TLB, L1/L2 Cache, RAM, and Swap Space latencies and utilization.
* **AI Memory Assistant:** An integrated chat interface (powered by Claude) to explain complex OS concepts or execute simulator commands.
* **Synth Sound Engine:** Custom audio feedback for memory operations using the Web Audio API.

## 💻 Tech Stack

* **Frontend:** HTML5, CSS3 (Neon-Cyberpunk UI), JavaScript (ES6+).
* **Graphics:** HTML5 Canvas for real-time particle effects.
* **Audio:** Web Audio API for interactive soundscapes.

## 📖 How to Use

1. **Paging Tab:** Use the "Allocate" buttons or click free frames to assign pages. Check the Address Translation box to see how offsets work.
2. **Segmentation Tab:** Drag the sliders to resize memory segments and observe how they fit into the 64KB address space.
3. **Virtual Tab:** Click through the hierarchy levels to see latency differences and learn about page replacement policies like LRU and FIFO.
4. **AI Assistant:** Click the robot icon in the bottom-left to ask questions or run a "Stress Test".
